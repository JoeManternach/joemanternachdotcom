Internet Protocol

Using python and our imagination my friend and I crafted an internet protocol. Using TCP we sent data packets, ACK's, and NACK's. To demonstrate our protocol we created a video game. We called this game "Plant Protocol". The player would start up the game and be prompted with the login screen. From there our protocol would be used for communication between the client and server. After the user logins in the client would send a communication to the server with the players username. The server would then send the client the players plants and other information. At the end the client sends the server the up to date version of the plant list and other information like hunger and thirst levels. In this project I learned how to setup server and client communications and also multithreading.
